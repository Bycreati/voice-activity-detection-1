# Voithru AI Voice Activity Detection (VAD)
This repository provides voice activity detection (VAD) dataset.
This dataset is used in SELF-ATTENTIVE VAD: CONTEXT-AWARE DETECTION OF VOICE FROM NOISE (ICASSP 2021)
